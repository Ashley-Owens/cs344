1. Unzip the assignment into your preferred directory
2. Navigate to the folder containing the unzipped project files: owensas_program2
3. Compile the program by entering either of the following two commands:
    a. gcc --std=gnu99 -o movies_by_year main.c
    b. gcc --std=gnu99 -g -Wall -o movies_by_year main.c
4. Run the program by entering the following command:    
    ./movies_by_year
