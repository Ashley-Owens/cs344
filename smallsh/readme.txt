Instructions for Running Program 3
1. Unzip the assignment into your preferred directory
2. Navigate to the folder containing the unzipped project files: owensas_program3
3. Compile the program by entering either one of the following two commands:
    a. gcc --std=gnu99 -o smallsh main.c
    b. gcc --std=gnu99 -g -Wall -o smallsh main.c
4. Run the program by entering the following command:       
    ./smallsh
